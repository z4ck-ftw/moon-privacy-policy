# Moon Music Bot – Privacy Policy

Last Updated: [26-11-2025]

Moon Music Bot (“the Bot”) is a Discord music bot designed to provide audio playback, queue management, and related features within Discord servers.
This Privacy Policy explains what data the Bot collects, how it is used, and how it is protected.

**1. Information We Collect**

Moon Music Bot does NOT collect any personal information such as real names, emails, phone numbers, IP addresses, or payment data.

The Bot only processes limited discord-related data necessary for its functionality:

**1.1. Temporary Data (Not Stored)**

These are processed in real time and immediately deleted after use:

Commands you send to the bot

Song names, URLs, search queries

Voice channel IDs for joining/leaving

Guild (server) IDs for managing music playback

**1.2. Optional Saved Data**

If the bot includes settings or presets, the following may be stored:

Server-specific configuration (prefix, DJ role, filters, etc.)

User-specific settings (if feature enabled)

No personal user identity or sensitive information is collected or stored.

**2. How We Use Your Data**

The collected data is used only to:

Execute bot commands

Manage music playback

Store and apply server settings

Improve bot performance and stability

We do not sell, share, or use data for marketing.

**3. Data Storage & Security**

Only essential configuration data is stored in a secure database.

All temporary command data is deleted immediately after execution.

We use standard security practices to prevent unauthorized access.

You may request deletion of your server’s stored config at any time.

**4. Third-Party Services**

Moon Music Bot may interact with services like:

YouTube

Spotify

SoundCloud

Discord API

These services have their own privacy policies. The Bot does not store data retrieved from these services.

**5. Children’s Privacy**

The Bot is intended for use only on Discord (13+ per Discord ToS).
We do not knowingly collect information from children under 13.

**6. Data Removal Requests**

Server owners can request:

Removal of server configuration

Reset of all saved settings

Contact the developer directly via Discord: [z4ck.ftw]

**7. Changes to This Policy**

Any changes will be posted on the official support server or bot documentation.
Continued use of the Bot means you accept the updated policy.
